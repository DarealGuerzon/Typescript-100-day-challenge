# 100 Days of TypeScript

A daily coding challenge to learn TypeScript and build toward full-stack development.

## About

One challenge per day, starting from the basics and progressing toward building full-stack applications.

## Tech Stack

- TypeScript
- Node.js
- ts-node

## Setup

```bash
npm install
```

## Run a Challenge

```bash
npx ts-node Day/Day1.ts
```

Replace `Day1.ts` with the file for any day.
